Image To Wave:

Program to convert an image file into a sound, whose spectrogram looks like the original image
------------------------------------------------------------------------------------------------------------------

right now only supports .wav and BITMAPINFOHEADER and BITMAPCOREHEADER from .bmp

.wav file format info from https://cs.wikipedia.org/wiki/WAV
.bmp file format info from https://en.wikipedia.org/wiki/BMP_file_format
